# Amrit Sugandh | Full-Stack Portfolio

A modern, high-performance portfolio website built with a focus on motion, aesthetics, and usability. Features a dynamic Admin Portal for code-free content management.

## 🚀 Live Demo
**[Launch Portfolio](https://portfolio-eight-tau-7d7gh5syd4.vercel.app/)** 

## ✨ Key Features
- **Dynamic Content**: Powered by Firebase Firestore.
- **Admin Portal**: Secure dashboard to update Profile, Projects, and Skills without touching code.
- **Glassmorphism UI**: Beautiful, translucent design language with vibrant gradients.
- **Interactive Motion**: Smooth GSAP animations and micro-interactions.
- **Responsive Design**: Optimized for all devices from mobile to ultra-wide displays.
- **WhatsApp Integration**: Real-time communication directly from the contact form.

## 🛠️ Technical Stack
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS.
- **Animations**: GSAP (GreenSock Animation Platform).
- **Backend/Database**: Firebase Firestore.
- **Auth**: Firebase Authentication.
- **Icons**: Phosphor Icons.

## 📂 Project Structure
- `index.html`: Main portfolio landing page.
- `admin.html`: Secure administrative dashboard.
- `script.js`: Core frontend logic and animations.
- `app-render.js`: Handles dynamic data fetching and rendering.
- `firebase-config.js`: Firebase configuration and initialization.

## 🔧 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/amritsugandh/portfolio.git
   ```
2. Open `index.html` in your browser.
3. For Admin Portal access, configure your Firebase Auth credentials.
